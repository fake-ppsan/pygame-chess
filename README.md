PYGAME CHESS

This is a fully playable two player chess game built from scratch in Python using Pygame.


This project started as a way to learn Python and game development during my summer vacations, and gradually evolved into a serious passion project, turning from just a simple piece moving game in the terminal into a complete chess engine with legal move generation, check/checkmate detection, special moves, move history, and a graphical interface built using Pygame.

FEATURES:

• Complete chess board rendered with Pygame
• Legal move generation for all pieces
• Check detection
• Checkmate detection
• Stalemate detection
• Pawn promotion(currently only queens supported)
• En passant
• Castling
• Move history with chess notation(disambiguation not supported)
• Captured piece display
• Visual indication when king is in check
• Piece sprites
• Legal move indicators

TECHNOLOGIES USED:

• Python 3
• Pygame

HOW TO RUN:

1. Clone the repository

>>> git clone https://github.com/fake-ppsan/pygame-chess.git

2. Install Pygame 

>>> pip install pygame

3. Run the game

>>> python chess.py
